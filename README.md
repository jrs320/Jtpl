# 类似vuejs 2.x单文件框架

    框架慢慢完善中，欢迎指正交流。
    支持浏览器版本：ie10以上，chrome，safari，firefox

## 安装使用

```
    npm i -S Jtpl
    npm i -D jtpl-loader
    npm i -D jtpl-sass-loader
```

## 使用说明

1. 必须通过webpack构建
2. 需要配置两个loader
    - jtpl-loader,
    - jtpl-sass-loader
3. 跟vuejs 2.x单文件使用类似
4. 只能存在一个跟元素
5. 标签必须要闭合
6. 在文件任何地方需要用到动态数据都使用 {{expression}} 表达式，指令除外
7. 具体使用可以参考项目中的example

## 已经实现的功能

1. 表达式求值
2. 嵌套子组件，父子组件传值
3. 组件css样式隔离，互不污染
4. 双向数据绑定
5. 数组操作的监听
6. 指令，目前实现了j-model，j-show，j-for
